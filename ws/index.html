О проекте

📋 Описание

Weather Seeker — это современный веб-сервис для получения точной и актуальной информации о погоде в любом регионе мира. Приложение сочетает в себе элегантный дизайн в стиле Liquid Glass (жидкое стекло) с мощными функциями для работы с погодными данными.

✨ Основные возможности

  • location_searching Автоматическое определение местоположения через браузер
  • search Поиск городов по названию (русский и английский языки)
  • cloud Детальная текущая погода: температура, ветер, влажность, давление, видимость, облачность
  • calendar_month Прогноз погоды на 5, 7, 10, 15 дней (до 16 дней через Open-Meteo)
  • animation Динамический фон с анимациями, соответствующими погодным условиям (дождь, снег, гроза, туман)
  • terrain Анимированный горный пейзаж, меняющий цвет в зависимости от погоды
  • cached Кэширование данных на 5 минут для быстрой загрузки
  • lightbulb Полезные советы по погоде для каждого дня
  • auto_stories Раздел интересных фактов о погоде с автоматическим обновлением
  • language Русскоязычный интерфейс

🛠️ Технологический стек

code JavaScript ES6+ (Modules, Async/Await, Fetch API)

🎨 Дизайн

Интерфейс выполнен в стиле Liquid Glass — современной вариации Glassmorphism с чёрно-белой цветовой палитрой. Эффект "жидкого стекла" создаётся с помощью:

  • Фоновых размытий (backdrop-filter: blur)
  • Полупрозрачных слоёв (rgba)
  • Тонких границ и теней
  • Плавных анимаций

Горный фон в нижней части экрана динамически меняет цветовую гамму в зависимости от текущей погоды, создавая атмосферное впечатление.

📊 Погодные параметры

Приложение отображает следующие данные:

  • Температура (°C) и ощущаемая температура
  • Скорость ветра (км/ч)
  • Влажность воздуха (%)
  • Давление (мм рт.ст.)
  • Видимость (км)
  • Облачность (%)
  • Координаты и локальное время
